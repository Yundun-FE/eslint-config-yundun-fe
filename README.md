# eslint-config-yundun-fe

The package provides a eslint's extensible shared config, be use for that project base on webpack & vue.

# Install

``` bash
npm install eslint-config-standard --save-dev
```

# Usage

add this to your .eslintrc file:

``` json
{
  "extends": "standard"
}
```
